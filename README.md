## 内容
- Android：機体は画面をタッチしている方向に進む。画面タッチしている間弾を出し続ける。
- PC：機体がマウスカーソルに追従する。右クリックで弾を発射。

## Platform
* Android
* PC

## 環境
- Unity2019.2.3f1 Personal
